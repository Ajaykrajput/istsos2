TestingÂ¶

The istSOS package cames with a TEST application, that can be load at the following url:

http://localhost/sos/testPOST/test.html

This page allows you to test requests in POST and GET mode and see the istSOS response.

Some predefined requests are already setted for your convenience, to fully test the service you could perform the follwing actions:

  1. Register a Sensor (RegisterSensor request)
  2. Insert some observations (InsertObservation request)
  3. Look at the service capabilities (GetCapabilities request)
  4. Look at a resgistered sensor details (DescribeSensor request)
  5. Get required observations (GetObservation request)
  6. Get the interested feature of interest (GetFeatureOfInterest request)
