=========================
Welcome to istSOS-project
=========================

*Free and Open Source Sensor Observation Service Data Management System*
   
.. figure::  images/istsos_logo.png
   :align:   center
   :scale:   100


istSOS is an OGC SOS_ server implementation written in Python_. 
istSOS allows for managing and dispatch observations from monitoring sensors 
according to the Sensor Observation Service standard. 

The project provides also a Graphical user Interface that allows for easing the 
daily operations and a RESTFull Web api for automatizing administration procedures.

istSOS is released under the GPL :ref:`license`, and runs on all major platforms 
(Windows, Linux, Mac OS X), even though tests were conducted under a linux environment.
